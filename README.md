# Лабораторная работа №5

## Описание
Данный проект представляет собой страницу из ЛР2, дополненную интерактивными элементами на JavaScript. Страница посвящена сообществу музыкантов **MusicVibe** и включает разделы: "О нас", "Новости", "Галерея" и "Контакты".

## Ответы на вопросы

**1. Какие интерактивности добавлены на страницу?**  
- Кнопка "Наверх" для быстрого возврата к началу страницы.  
- Аккордеон в разделе "Наша команда", раскрывающий описание каждого участника.  
- Фотогалерея с фильтрацией по категориям "Studio" и "Concert".  
- Модальное окно для просмотра увеличенных изображений из галереи.  
- Навигация между секциями с плавным переключением.  

**2. Где использовался ИИ, а где код писался вручную?**  
- **ИИ:** помощь в написании скриптов фильтрации, модального окна и аккордеона, советы по структурированию HTML/CSS и JS.  
- **Вручную:** весь HTML, CSS, подключение видео и изображений, базовые элементы страницы (шапка, футер, контент).  

**3. Что показалось самым интересным или сложным?**  
- Интересно было реализовать фильтрацию галереи с правильными категориями и индексацией изображений.  
- Сложнее всего было корректно объединить модальное окно с фильтрацией, чтобы открывались только видимые изображения.  

**4. Какие запросы делали LLM?**  
- Добавление кнопки "Наверх" с плавной прокруткой.  
- Написание модального окна для увеличения фото.  
- Советы по структурированию кода и стилям CSS.
